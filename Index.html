Halo! Nama saya Revalda, dan saya saat ini sedang mempelajari dunia Front-End Development. Saya tertarik dengan bagaimana sebuah website dapat terlihat menarik, responsif, dan interaktif bagi pengguna. Oleh karena itu, saya terus belajar berbagai teknologi yang digunakan dalam pengembangan antarmuka web.

Development

Website ini dibuat menggunakan HTML, CSS, dan JavaScript

HTML digunakan sebagai dasar struktur halaman, CSS berfungsi untuk mempercantik desain dan tata letak, sementara JavaScript digunakan untuk menambahkan interaktivitas dan fungsionalitas dinamis. Kombinasi ketiga teknologi ini memastikan website dapat berjalan dengan baik dan memberikan pengalaman pengguna yang optimal.
